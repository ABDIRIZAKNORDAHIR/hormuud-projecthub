Double-click OPTION_B_START.bat to open this file + Supabase + GitHub + Render.


STEP 1 — SUPABASE (cloud database)                    ~10 minutes
──────────────────────────────────────────────────
1. Go to https://supabase.com → Sign up (free)
2. New project
   • Name: projecthub
   • Password: pick one and WRITE IT DOWN
   • Region: Frankfurt or Mumbai (closest to you)
3. Wait until project is ready (green)
4. Settings (gear) → Database → Connection string → URI
5. Copy the URI
6. Replace [YOUR-PASSWORD] with your real password
7. Paste into Notepad — label it DATABASE_URL

   Example shape:
   postgresql://postgres.xxxxx:YOUR_PASSWORD@aws-0-eu-central-1.pooler.supabase.com:6543/postgres


STEP 2 — GITHUB (upload your code)                    ~15 minutes
──────────────────────────────────────────────────
IMPORTANT: .gitignore is set — your .env file will NOT be uploaded (safe).

1. Install GitHub Desktop: https://desktop.github.com
2. Create free account at github.com if needed
3. GitHub Desktop → File → Add local repository
   Folder: C:\Users\25261\Downloads\Design ProjectHub UI_UX
4. If asked → Create repository → name: hormuud-projecthub
5. Click "Publish repository" → Publish
6. Remember your GitHub username (e.g. ahmed123)


STEP 3 — RENDER (host the app 24/7)                   ~10 minutes
──────────────────────────────────────────────────
1. Go to https://render.com → Sign up with GitHub
2. New + → Blueprint
3. Connect GitHub → select hormuud-projecthub repo
4. Click Apply (Render reads render.yaml automatically)
5. Click service "hormuud-projecthub" → Environment tab
6. Add these TWO variables (click Add Environment Variable):

   Key: DATABASE_URL
   Value: (paste Supabase URI from Step 1)

   Key: GROQ_API_KEY
   Value: (open your .env file in project folder — copy GROQ_API_KEY line value)
          Or run SETUP_FREE_AI.bat if you need a new key

7. Save Changes → wait for deploy (5–15 min, status = Live)
