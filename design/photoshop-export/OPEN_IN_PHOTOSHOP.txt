WHERE TO OPEN
-------------
1. ALL SCREENS IN ONE FILE (recommended)
   ProjectHub-Full-Design.psb

   → Each screen is a separate layer, stacked vertically

   → Same design, split into two Photoshop files

   screens\01-welcome-home.psd

FREE ALTERNATIVE
----------------
Upload any .psd file to https://www.photopea.com (works in browser)

REGENERATE
----------
Double-click: EXPORT_FULL_PSD.bat (project root)
Or run: npm run export:figma-screens && npm run export:photoshop
(ProjectHub must be running on http://localhost:5180)
